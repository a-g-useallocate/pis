# StylModa v3 — production release

Учебная информационная система интернет-магазина одежды StylModa.

## Что входит в версию

- десктопное приложение на Python/Tkinter;
- SQLite-база с начальными пользователями, товарами, заказами и результатами релизных исправлений;
- роли: администратор, менеджер, покупатель;
- каталог, фильтрация, избранное, корзина, заказы, статусы, обратная связь;
- усиленная валидация после β-тестирования;
- журналирование критических ошибок в `logs/stylmoda_production.log`;
- лендинг в каталоге `landing`;
- инструкция production-размещения в `deploy/README_PRODUCTION.md`.

## Запуск

```bash
python main.py
```
