# CHANGELOG v3

## Исправления после β-тестирования

1. Добавлена расширенная проверка регистрационных данных: логин, телефон, e-mail.
2. Усилена проверка заказа: адрес обязателен для доставки, комментарий ограничен по длине.
3. Обновлены тексты интерфейса: версия позиционируется как production-релиз после β-тестирования.
4. Добавлен журнал критических ошибок запуска.
5. Подготовлены файлы production-размещения и сборки.
6. Разработана лендинговая страница системы.
