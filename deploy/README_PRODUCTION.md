# Размещение StylModa v3 в production/test окружении

Для десктопной версии в качестве production-окружения используется сборка исполняемого файла и публикация релиза на GitHub Releases. Лендинг размещается отдельно как статическая страница GitHub Pages.

## 1. Публикация проекта на GitHub

1. Создать репозиторий `stylmoda-v3`.
2. Загрузить все файлы проекта.
3. В настройках репозитория включить Pages: `Settings → Pages → Deploy from branch → main → /landing`.
4. После публикации открыть адрес GitHub Pages и проверить лендинг.

## 2. Сборка приложения

На Windows выполнить:

```bat
deploy\build_windows.bat
```

Результат сборки появится в каталоге `dist`.
