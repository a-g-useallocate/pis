Общие сведения

Информационная система для управления продажами одежды

Система предназначена для автоматизации работы интернет‑магазина StylModa. Покупатель выбирает товары и оформляет заказ, менеджер контролирует заявки и остатки, администратор управляет каталогом, пользователями и результатами обратной связи.

Для покупателя

Регистрация, просмотр каталога, фильтрация товаров, избранное, корзина, оформление заказа и отслеживание статуса.

Для менеджера

Просмотр заказов, изменение статусов, контроль остатков, обработка замечаний по результатам тестирования.

Для администратора

Управление товарами, пользователями, ролями, журналами и релизными исправлениями после β‑тестирования.

Что исправлено в v3

Релиз подготовлен с учетом критических замечаний β‑тестирования

  • усилена проверка логина, телефона, e‑mail, адреса доставки и комментариев;
  • добавлен production‑контур запуска с журналом ошибок;
  • подготовлена структура для публикации на GitHub Releases и GitHub Pages;
  • оформлен отдельный лендинг для десктопной версии системы.
